EngineMode

public enum EngineMode: TessOcrEngineMode.RawValue

Specifically determines the underlying method that Tesseract uses to perforn OCR

  • tesseractOnly: The legacy Tesseract engine. This can only use training data from the tessdata repository

  • lstmOnly: Utilizes a long short-term memory recurrent neural network. This can use training data from tessdata, tessdata_best, or tessdata_fast respositories

  • tesseractLstmCombined: A combination of the legacy Tesseract engine and a long short-term memory recurrent neural network. This can only use training data from the tessdata repository
